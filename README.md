# SimonSay ArduinoiUno
This is a simple sketch of the game Simon on arduino nano. 
The circuit was simulated using the Wokwi platform.
## Components
- Arduino Nano
- 4 x PushButton
- 4 x Led
- 4 x Resistor 220 Omh
- Buzzer
## Libs
- ArduinoSTL by Mike Matera
